## .swf file (Pizzatron.swf) games

1- swfdump -D game.swf

2- extract scripts and data

wget https://github.com/jindrapetrik/jpexs-decompiler/releases/download/version18.5.0/ffdec_18.5.0.zip
unzip ffdec_18.5.0.zip
cd ffdec_18.5.0
java -jar ffdec.jar


3- run the game
